# Debugging Linux

## Objetivo

Investigar problemas sem destruir evidências.

## Ordem segura

1. coletar sintomas;
2. checar logs;
3. checar recursos;
4. checar rede;
5. reproduzir com segurança;
6. aplicar mitigação reversível;
7. registrar causa.
