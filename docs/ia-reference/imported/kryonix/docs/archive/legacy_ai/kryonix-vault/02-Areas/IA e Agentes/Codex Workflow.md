# Codex Workflow

## Objetivo

Padronizar tarefas para Codex/LLMs com segurança e previsibilidade.

## Fluxo recomendado

1. Escrever issue pequena.
2. Definir escopo e fora do escopo.
3. Referenciar arquivos prováveis.
4. Exigir testes.
5. Exigir menor diff correto.
6. Revisar PR manualmente.
7. Rodar CI.
8. Atualizar docs se necessário.
