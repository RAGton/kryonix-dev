# Playbook - Analise de Logs

## Objetivo

Transformar logs em hipótese verificável.

## Procedimento

1. Definir janela de tempo.
2. Filtrar por request/job id.
3. Separar erro causa de erro consequência.
4. Procurar primeira falha.
5. Correlacionar com deploy/mudança.
6. Gerar hipótese e validação.
