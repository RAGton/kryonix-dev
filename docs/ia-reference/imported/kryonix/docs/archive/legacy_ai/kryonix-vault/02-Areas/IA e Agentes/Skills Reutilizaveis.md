# Skills Reutilizaveis

## Objetivo

Transformar workflows repetidos em instruções reutilizáveis para reduzir prompt longo.

## Quando criar Skill

- tarefa se repete;
- tem checklist fixo;
- precisa de formato previsível;
- economiza contexto;
- reduz erro do agente.
