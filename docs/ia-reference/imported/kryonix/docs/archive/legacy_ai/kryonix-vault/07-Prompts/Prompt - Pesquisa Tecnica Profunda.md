# Prompt - Pesquisa Tecnica Profunda

```txt
Atue como pesquisador técnico sênior.

Tema: <tema>

Objetivo: produzir uma pesquisa prática, confiável e aplicável em produção.

Obrigatório:
- priorize fontes oficiais e materiais reconhecidos;
- separe fato confirmado, boa prática, opinião técnica, hipótese e risco;
- explique trade-offs;
- inclua exemplos reais;
- inclua checklist de decisão;
- não recomende hype sem utilidade prática.

Formato:
1. Resumo executivo
2. Contexto
3. Opções existentes
4. Trade-offs
5. Riscos
6. Recomendação prática
7. Checklist
8. Fontes
```
