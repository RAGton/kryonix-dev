# OWASP API Top 10

## Objetivo

Usar riscos OWASP como checklist prático para APIs.

## Pontos recorrentes

- autorização por objeto;
- autenticação quebrada;
- autorização por propriedade;
- consumo irrestrito de recursos;
- exposição excessiva de dados;
- SSRF;
- configuração incorreta;
- inventário de APIs;
- uso inseguro de APIs terceiras.
