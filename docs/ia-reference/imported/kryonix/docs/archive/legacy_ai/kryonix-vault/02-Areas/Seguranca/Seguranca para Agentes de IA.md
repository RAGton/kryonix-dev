# Seguranca para Agentes de IA

## Objetivo

Limitar risco operacional ao usar agentes.

## Regras

- sandbox;
- escopo pequeno;
- sem secrets;
- sem produção direta;
- sem rede quando não precisa;
- revisão humana;
- PR pequeno;
- logs de ação.
