# Playbook - Transformar Ideia em MVP

## Objetivo

Reduzir ideia a produto mínimo real.

## Perguntas

1. Quem usa?
2. Que dor resolve?
3. Qual ação principal?
4. O usuário pagaria ou usaria semanalmente?
5. Qual dado precisa persistir?
6. Qual risco de segurança?
7. Como medir valor?

## Entrega

- escopo MVP;
- fora do escopo;
- API principal;
- modelo de dados;
- plano de validação.
