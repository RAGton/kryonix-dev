# MVP Vendavel

## Objetivo

Construir menor produto útil que alguém pagaria ou usaria de verdade.

## Deve ter

- problema claro;
- usuário definido;
- fluxo principal;
- métrica de valor;
- suporte básico;
- deploy estável;
- documentação mínima.
