# Estrategia de Custo de Tokens

## Objetivo

Reduzir custo com LLM mantendo qualidade.

## Técnicas

- contexto fixo em notas;
- workflows em Skills;
- prompts reutilizáveis;
- issues pequenas;
- resumos de projeto;
- não reenviar arquivos grandes;
- usar modelos menores para tarefas simples;
- usar modelos fortes só para arquitetura, segurança e bugs difíceis.

## Checklist

- [ ] O contexto fixo está em uma nota?
- [ ] Existe Skill para esse workflow?
- [ ] A tarefa foi quebrada?
- [ ] A saída esperada está clara?
- [ ] O modelo usado é adequado ao risco?
