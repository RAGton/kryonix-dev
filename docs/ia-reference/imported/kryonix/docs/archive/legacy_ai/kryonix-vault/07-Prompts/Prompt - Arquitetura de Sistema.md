# Prompt - Arquitetura de Sistema

```txt
Atue como arquiteto sênior de software.

Projete a arquitetura para: <sistema>

Contexto:
- usuário alvo: <quem>
- escala esperada: <escala>
- restrições: <restrições>
- stack preferida: <stack>
- risco principal: <risco>

Entregue:
- requisitos funcionais;
- requisitos não funcionais;
- desenho de módulos;
- modelo de dados;
- APIs/contratos;
- autenticação/autorização;
- observabilidade;
- segurança;
- deploy/rollback;
- trade-offs;
- riscos;
- MVP recomendado;
- o que NÃO fazer agora.
```
