# Graph + Vector Hybrid Search

Esta nota trata da integração entre busca semântica em grafos (LightRAG) e busca vetorial pura.

## Status
- **ROADMAP**: Implementação pendente na API central.

## Objetivo
Unificar a precisão do grafo (entidades e relações) com a flexibilidade do vetor (similaridade de cosseno) para respostas mais precisas.
