# Playbook - Configurar Repo para Codex

## Objetivo

Preparar um repo para agentes trabalharem com segurança.

## Procedimento

1. Criar `AGENTS.md`.
2. Criar `ai/INDEX.md` e `docs/README.md`.
3. Documentar comandos de teste em `docs/TESTING.md`.
4. Criar issues pequenas.
5. Proteger secrets.
6. Garantir CI (docs-audit.yml).
7. Definir fora do escopo.
8. Exigir PR pequeno.
