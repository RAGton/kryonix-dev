# Hardening de Servicos

## Objetivo

Reduzir superfície de ataque de serviços.

## Aplicar

- usuário sem privilégios;
- filesystem restrito;
- variáveis seguras;
- limites de recursos;
- firewall;
- TLS;
- logs sem segredo;
- atualização controlada.
