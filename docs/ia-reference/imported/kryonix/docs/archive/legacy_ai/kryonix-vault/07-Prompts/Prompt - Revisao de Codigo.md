# Prompt - Revisao de Codigo

```txt
Atue como revisor sênior de código.

Revise o diff abaixo com foco em:
- corretude;
- segurança;
- testes;
- legibilidade;
- performance;
- compatibilidade;
- escopo;
- manutenção.

Classifique achados como:
- bloqueante;
- importante;
- sugestão.

Não peça refatoração estética sem benefício claro.

Diff/contexto:
<conteúdo>
```
