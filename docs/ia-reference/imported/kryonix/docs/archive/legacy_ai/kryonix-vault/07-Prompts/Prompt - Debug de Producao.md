# Prompt - Debug de Producao

```txt
Atue como engenheiro SRE.

Sintomas:
<sintomas>

Logs/métricas:
<dados>

Restrições:
- não sugerir ação destrutiva sem backup;
- priorizar diagnóstico seguro;
- separar hipótese de evidência;
- incluir rollback.

Entregue:
1. impacto provável
2. hipóteses ordenadas
3. comandos seguros de diagnóstico
4. mitigação reversível
5. correção definitiva
6. pós-incidente
```
