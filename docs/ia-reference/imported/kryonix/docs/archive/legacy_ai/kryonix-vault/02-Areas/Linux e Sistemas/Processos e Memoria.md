# Processos e Memoria

## Objetivo

Entender execução de programas no Linux.

## Conceitos

- processo;
- thread;
- PID;
- sinais;
- memória virtual;
- file descriptors;
- limites do sistema.

## Comandos úteis

```bash
ps aux
top
htop
pidstat
pmap
ulimit -a
```
