# Playbook - Revisar Codigo Gerado por IA

## Objetivo

Evitar aceitar código plausível, mas errado.

## Checklist

- [ ] O diff é pequeno?
- [ ] O escopo foi respeitado?
- [ ] Eu entendo a mudança?
- [ ] Tem teste?
- [ ] Quebrou contrato público?
- [ ] Autorização está correta?
- [ ] Validação está na borda?
- [ ] Logs vazam dados?
- [ ] Dependência nova é necessária?
- [ ] Rollback é simples?
