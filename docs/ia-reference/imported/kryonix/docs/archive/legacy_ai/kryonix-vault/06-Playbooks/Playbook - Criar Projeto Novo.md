# Playbook - Criar Projeto Novo

## Objetivo

Iniciar projeto vendável com base técnica limpa.

## Procedimento

1. Definir problema real e público alvo.
2. Criar `docs/README.md` e `docs/ARCHITECTURE.md`.
3. Criar `ai/INDEX.md` e `AGENTS.md`.
4. Criar stack mínima (NixOS Flake se possível).
5. Criar `docs/TESTING.md` e testes iniciais.
6. Criar `.github/workflows/` para CI.
7. Criar OpenAPI se for API.
8. Criar `.env.example`.
9. Criar plano de deploy e rollback em `docs/OPERATIONS.md`.

## Checklist

- [ ] problema claro
- [ ] usuário alvo
- [ ] MVP limitado
- [ ] testes
- [ ] CI
- [ ] segurança mínima
- [ ] docs mínimas
