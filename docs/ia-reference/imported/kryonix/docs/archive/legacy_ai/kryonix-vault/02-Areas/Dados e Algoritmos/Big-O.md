# Big-O

## Objetivo

Estimar crescimento de custo em CPU/memória.

## Prática backend

Importa muito em:

- loops sobre muitos registros;
- queries sem índice;
- serialização grande;
- cache mal usado;
- processamento de logs;
- filas com backlog.
