## Regras para IA

1. Consultar este vault antes de responder.
2. Priorizar notas locais sobre memória interna.
3. Diferenciar fato, hipótese, opinião técnica e decisão.
4. Não inventar comando, flag, recurso ou compatibilidade.
5. Quando houver risco operacional, pedir validação humana.
6. Gerar código com teste, validação, logs seguros e menor diff correto.
