# LLM como Amplificador

## Objetivo

Usar IA para acelerar engenharia sem terceirizar julgamento técnico.

## Regra prática

A IA pode sugerir. O humano decide, valida e assume responsabilidade.

## Usar IA para

- resumir código;
- gerar testes;
- revisar PR;
- criar documentação;
- pesquisar alternativas;
- decompor tarefas;
- criar boilerplate;
- simular trade-offs.

## Não usar IA sozinha para

- autorização;
- criptografia;
- secrets;
- deploy produção;
- migrations destrutivas;
- arquitetura crítica;
- billing;
- segurança de multi-tenant.
