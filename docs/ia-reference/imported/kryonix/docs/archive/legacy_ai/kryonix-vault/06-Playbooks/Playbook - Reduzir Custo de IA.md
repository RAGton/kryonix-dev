# Playbook - Reduzir Custo de IA

## Objetivo

Diminuir gasto com tokens e chamadas.

## Procedimento

1. Identificar prompts repetidos.
2. Criar Skills.
3. Criar notas de contexto fixo.
4. Criar `PROJECT_SUMMARY.md` por projeto.
5. Dividir tarefas.
6. Usar modelo barato em tarefas simples.
7. Usar modelo forte só em risco alto.
