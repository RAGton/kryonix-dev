# Arquitetura Backend Profissional

## Objetivo

Criar backend seguro, testável, observável e vendável.

## Componentes mínimos

- autenticação;
- autorização;
- validação de entrada;
- contratos OpenAPI;
- banco com migrations;
- logs estruturados;
- rate limit;
- tratamento de erros;
- testes;
- CI;
- deploy com rollback.
