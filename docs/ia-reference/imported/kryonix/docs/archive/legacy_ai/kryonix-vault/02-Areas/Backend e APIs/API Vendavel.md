# API Vendavel

## Objetivo

API que pode ser usada por clientes reais sem depender do desenvolvedor ao lado.

## Exige

- documentação clara;
- versionamento;
- erros consistentes;
- autenticação;
- limites de uso;
- estabilidade de contrato;
- exemplos;
- monitoramento;
- suporte a idempotência em writes importantes.
