# Prompt - Revisao de Seguranca

```txt
Atue como engenheiro de segurança de aplicações.

Revise o sistema/código/API com foco em:
- autenticação;
- autorização;
- object-level authorization;
- validação de entrada;
- abuso de recursos;
- secrets;
- logs;
- SSRF;
- injeção;
- multi-tenancy;
- CI/CD;
- deploy.

Entregue:
- achados por severidade;
- evidência;
- impacto;
- correção mínima;
- teste de validação;
- risco residual.
```
