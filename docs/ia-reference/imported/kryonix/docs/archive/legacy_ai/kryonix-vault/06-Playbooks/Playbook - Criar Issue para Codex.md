# Playbook - Criar Issue para Codex

## Objetivo

Criar tarefa pequena, segura e verificável para agente.

## Estrutura

- objetivo;
- contexto;
- escopo;
- fora do escopo;
- arquivos prováveis;
- critérios de aceite;
- validação;
- riscos.

## Regra

Uma issue deve gerar um PR pequeno.
